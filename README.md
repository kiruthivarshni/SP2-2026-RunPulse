## Project Explanation

RunPulse is an AI-powered IPL PowerPlay score prediction model developed for SONA POWER PREDICT 2026. The project analyzes historical IPL match data, player performance, team statistics, venue conditions, and bowling metrics to accurately predict PowerPlay scores.

The model is built using Gradient Boosting Regressor from scikit-learn and uses feature engineering techniques such as rolling averages, recent team form, head-to-head analysis, venue-based statistics, and opener pair performance to improve prediction accuracy and reliability.
